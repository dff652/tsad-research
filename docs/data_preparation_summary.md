## 五、关键发现与建议

### 关键发现

1. **Timer 显著领先**：均分 0.672，通过率 83.6%，远超 ChatTS (0.336) 和 Qwen-VL-8B (0.225)
2. **Qwen-VL-8B 表现最差**：68.2% 差评率，仅在通信/系统类传感器上表现突出
3. **评审人分歧较大**：Timer 的 Fleiss' Kappa 仅 0.175，说明"好"与"一般"的边界模糊
4. **67% 点位评价为"效果好"**：但不同模型在这些点位上的得分差异很大
5. **过检场景**：ChatTS 在过度检测场景上得分最高(0.575)，Timer反而较低(0.525)
6. **Current/Power 类异常率最高(27.51%)**，Timer 在该类上得分最高(0.84)

### 后续建议

1. **以 Timer 作为 baseline**：在 Benchmark 中重点对标 Timer
2. **分析 ADTK_HBOS 的推理图片**：对比人工评价与算法标注的差异
3. **扩展评分维度**：建议增加检测延迟、漏检率、误报率等细分维度
4. **利用全量数据**：436个点位的推理特征可用于弱监督质量预测
